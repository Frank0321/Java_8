## Lambda
- 主要用來實作interface method(方法定義)使用匿名函式使的語法上更精簡!
  - 匿名函數也就是沒有名稱的方法
  - @FunctionalInterface : 單一函式介面 annotation
- 必須包含三個部分 : 
  1. 方法參數
  2. 箭頭符號
  3. 方法內容
- 透過把函式作為方法的參數傳遞，使的程式碼變的更加簡潔
